## Get Started

```bash
pnpm install # install dependencies

pnpm dev # start local server
```

## Реалізований функціонал

- Лічильник виконаних завдань

- Форма для створювання нового завдання

- Список доступних фільтрів (всі, активні та невиконані завдання)

- Список завдань

- Можливість редагування тексту завдання

- Можливість відмітити завдання виконаним, або невиконаним

- Локальне збереження завдань

## Додав би, якщо б мав додатковий час:

- Виніс маніпулювання данними в стм або контекст, для уникнення проп дріллінгу

- Відрефакторив всі компоненти, більш явно розбивши функціонал

- Лоадер для відображення стану завантаження
